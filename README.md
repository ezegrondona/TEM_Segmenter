# TEM Segmenter

## Descripción

TEM Segmenter es una aplicación de escritorio desarrollada en Python para asistir en la segmentación de imágenes de Microscopía Electrónica de Transmisión (TEM).

El objetivo es combinar Inteligencia Artificial (MobileSAM) con la experiencia del investigador, permitiendo una segmentación rápida, precisa y compatible con Fiji/ImageJ.

---

## Filosofía del proyecto

> La inteligencia artificial propone.
> El investigador decide.

---

## Características previstas

- Lectura de imágenes TIFF, PNG y JPG.
- Lectura automática de calibración desde TIFF (ImageJ/Fiji).
- Calibración manual cuando sea necesario.
- Preprocesamiento inteligente.
- Segmentación asistida mediante MobileSAM.
- Aprobación interactiva de estructuras.
- Exportación compatible con Fiji ROI Manager.
- Exportación de máscaras TIFF.
- Exportación de mediciones en Excel.
